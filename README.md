# Paper-Glance-Demography

A three-step AI-assisted workflow for reading academic papers in demography, migration studies, and quantitative social science.

## Workflow

1. `paper-matrix-extract`  
   Extracts structured information from a single academic paper into a literature matrix.

2. `literature-pattern-analysis`  
   Analyzes a literature matrix to identify dominant theories, methods, variables, and research gaps.

3. `paper-research-core`  
   Provides deep structured reading notes for important papers, focusing on research question, theory, data, methods, findings, contribution, and limitations.

## Repository Structure

- `/skills` — reusable academic reading skills
- `/examples` — sample outputs
- `/templates` — reusable matrix and reading templates

## Intended Use

This repository is designed for literature review, thesis preparation, demographic research, migration studies, and social science research workflows.
